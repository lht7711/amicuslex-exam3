## 프로젝트 수행 방법

React App 구동을 위한 의존성 모듈을 설치:

```shell
npm install
```
---

의존성 패키지 모듈을 설치 후 `React app` 실행
### `npm start`

브라우저에서 [http://localhost:3000](http://localhost:3000) 으로 App이 정상적으로 실행되는것을 확인 할 수 있습니다.
